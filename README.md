# HDFC Stock Data Analysis 
HDFC Stock Analysis Dashboard (Power BI)

📌 Project Overview
This project presents an interactive stock analysis dashboard for HDFC Bank, built using Power BI. The dashboard provides deep insights into stock performance using key financial indicators such as price trends, trading volume, volatility, and market activity.

The goal of this project is to transform raw stock market data into meaningful visual insights that help in understanding market behavior, trends, and investment patterns.

📂 Data Source
The dataset used in this project is extracted from the National Stock Exchange (NSE) India official website.
Source: NSE India Historical Data
Data Type: Equity Historical Data (HDFC Bank)
Frequency: Daily
Fields Included:
Date
Open Price
High Price
Low Price
Close Price
Previous Close
Volume
VWAP (Volume Weighted Average Price)
Number of Trades
52 Week High
52 Week Low

📊 Dashboard Features
🔹 KPI Summary
Average 52 Week High & Low
Average Open & Close Price
Average Volume
Quick overview of stock performance
🔹 Monthly Analysis
Monthly Volume Distribution
Identifies high trading activity months
Helps detect seasonal trends
🔹 Quarterly Insights
52 Week High & Low comparison across quarters
Highlights market fluctuations over time
🔹 Combined Metrics Analysis
Volume, VWAP, and % Change trends
Shows relationship between price movement and trading activity
🔹 Trading Activity Analysis
Average number of trades by Year & Month
Helps understand market participation

📈 Key Insights
1. HDFC stock shows stable growth patterns with moderate volatility.
2. High trading volumes in certain months indicate increased investor interest.
3. The stock operates within a well-defined 52-week range, reflecting controlled price movement.
4. Trading activity has increased over time, suggesting strong market liquidity.
5. VWAP trends align closely with price movement, indicating efficient price discovery.

🛠️ Tools & Technologies Used
Power BI – Data visualization & dashboard creation
Excel / CSV – Data preprocessing
DAX (Data Analysis Expressions) – Calculations & measures

🎨 Dashboard Design
The dashboard is designed with a financial analytics theme, featuring:
1. Clean and structured layout
2. Interactive filters (Year, Quarter, Month, Day)
3. Consistent color scheme for better readability
4. KPI cards for quick insights

🚀 How to Use
1. Download the .pbix file
2. Open in Power BI Desktop
3. Use filters (Year, Month, Quarter) to explore data
4. Hover over visuals for detailed insights

📌 Conclusion

This dashboard provides a comprehensive view of HDFC stock performance by combining price trends, trading activity, and volume analysis. It serves as a useful tool for data analysts, finance students, and investors to understand stock behavior and derive insights for decision-making.

⭐ Future Enhancements
1. Add real-time stock data integration
2. Include moving averages & technical indicators
3. Implement predictive analytics
4. Add comparison with other banking stocks
